# CSS3 Advanced Selectors & Specificity

Master CSS selectors and the cascade. Learn pseudo-classes, pseudo-elements, attribute selectors, and combinators. Understand specificity, inheritance, and the cascade algorithm. Advanced selectors enable precise targeting and maintainable stylesheets.

|Topic|Focus & Purpose|
|---|---|

|**[[CSS3.3 Pseudo-Elements]]**|::before, ::after; ::first-letter, ::first-line; content property; generated content; decorative elements. Content enhancement.|
|**[[CSS3.4 Attribute Selectors]]**|[attribute]; [attribute="value"]; [attribute^="value"]; [attribute$="value"]; [attribute*="value"]; complex matching. Attribute targeting.|
|**[[CSS3.5 Specificity]]**|Specificity calculation; inline > ID > class > element; !important; specificity rules; specificity conflicts. Cascade rules.|
|**[[CSS3.6 Cascade & Inheritance]]**|Cascade algorithm; inheritance; inherited properties; non-inherited properties; controlling inheritance. CSS fundamentals.|
|**[[CSS3.7 CSS Variables]]**|Custom properties; --variable-name; var() function; scope; fallback values; dynamic theming. Reusable values.|
|**[[CSS3.8 Project: Theme System]]**|Building theme with variables; color schemes; reusable patterns; specificity management; maintainable CSS. Scalable styling.|
