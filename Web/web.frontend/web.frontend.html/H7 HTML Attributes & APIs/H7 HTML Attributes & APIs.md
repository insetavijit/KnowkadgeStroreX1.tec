# H7 HTML Attributes & APIs

Master HTML attributes and browser APIs. Learn global attributes, ARIA, microdata, and Web APIs. Understand attribute syntax, boolean attributes, and data binding. Attributes and APIs extend HTML functionality and enable rich web experiences.

| Topic                                   | Focus & Purpose                                                                                                                          |
| --------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **[[H7.1 Global Attributes]]**           | id, class, style; title, lang, dir; hidden, contenteditable; tabindex; draggable; data attributes. Universal attributes.                 |
| **[[H7.2 Boolean Attributes]]**          | Boolean syntax; common boolean attributes; disabled, readonly, required; checked, selected; attribute presence. True/false values.       |
| **[[H7.3 ARIA Basics]]**                 | Accessibility roles; aria-label, aria-labelledby; aria-describedby; aria-live; landmark roles; ARIA best practices. Accessibility layer. |
| **[[H7.4 Microdata & Schema.org]]**      | itemscope, itemtype, itemprop; structured data; rich snippets; SEO benefits; common schemas; implementation. Semantic metadata.          |
| **[[H7.5 Open Graph & Meta Tags]]**      | Social media meta tags; og:title, og:description; Twitter cards; meta tags for SEO; viewport; charset. Metadata.                         |
| **[[H7.6 Performance Attributes]]**      | loading="lazy"; async and defer; preload, prefetch; importance; resource hints; performance optimization. Speed enhancement.             |
| **[[H7.7 Security Attributes]]**         | rel="noopener noreferrer"; sandbox; CSP; crossorigin; integrity; security best practices. Protection.                                    |
| **[[H7.8 Project: SEO Optimized Page]]** | Complete page with metadata; structured data; Open Graph tags; accessibility attributes; performance optimization. Production-ready.     |
