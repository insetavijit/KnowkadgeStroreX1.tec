# H2 Lists & Tables

Master organizing content with HTML lists and tables. Learn ordered lists, unordered lists, definition lists, and data tables. Understand semantic markup for structured content. Lists and tables are fundamental for presenting organized information and are used extensively across the web.

| Topic                                  | Focus & Purpose                                                                                                                            |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **[[H2.1 Unordered Lists]]**            | ul element; li items; nested lists; list styling basics; use cases; accessibility. Bullet point lists.                                     |
| **[[H2.2 Ordered Lists]]**              | ol element; numbering types; start attribute; reversed lists; nested ordered lists; semantic numbering. Numbered lists.                    |
| **[[H2.3 Definition Lists]]**           | dl, dt, dd elements; terms and definitions; use cases; semantic markup; styling considerations. Glossaries and dictionaries.               |
| **[[H2.4 Table Basics]]**               | table element; tr, td, th; table structure; table headers; simple tables; when to use tables. Tabular data.                                |
| **[[H2.5 Advanced Tables]]**            | colspan and rowspan; thead, tbody, tfoot; caption; colgroup; complex tables; table accessibility. Structured data tables.                  |
| **[[H2.6 Table Accessibility]]**        | scope attribute; headers attribute; aria labels; screen reader support; accessible table markup. Inclusive tables.                         |
| **[[H2.7 Styling Lists & Tables]]**     | CSS basics for lists; list-style properties; table borders; spacing; zebra striping; responsive tables. Visual presentation.               |
| **[[H2.8 Project: Data Presentation]]** | Creating a resume with lists; building a schedule table; organizing information; semantic markup; accessibility practices. Real-world use. |
