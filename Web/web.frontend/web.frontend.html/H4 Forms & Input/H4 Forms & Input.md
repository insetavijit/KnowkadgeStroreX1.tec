# H4 Forms & Input

Master HTML forms for user interaction and data collection. Learn form elements, input types, validation, and accessibility. Build contact forms, registration forms, and complex data entry interfaces. Forms are essential for user engagement, e-commerce, and web applications.

| Topic                              | Focus & Purpose                                                                                                                          |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **[[H4.1 Form Basics]]**            | form element; action and method; input basics; labels; submit buttons; form structure; GET vs POST. Foundation.                          |
| **[[H4.2 Input Types]]**            | text, email, password, number; tel, url, search; date, time, color; file; range; input attributes. User input.                           |
| **[[H4.3 Form Controls]]**          | textarea; select and option; checkbox; radio buttons; button types; fieldset and legend; form grouping. Interactive elements.            |
| **[[H4.4 HTML5 Validation]]**       | required attribute; pattern attribute; min, max, maxlength; email and url validation; built-in validation; browser support. Client-side. |
| **[[H4.5 Labels & Accessibility]]** | label element; for attribute; implicit vs explicit labels; fieldset and legend; ARIA attributes; screen reader support. Inclusive forms. |
| **[[H4.6 Advanced Form Features]]** | datalist; autocomplete; placeholder; autofocus; multiple; form attributes; disabled and readonly. Enhanced UX.                           |
| **[[H4.7 Form Styling Basics]]**    | Styling inputs; focus states; validation states; custom checkboxes; button styling; form layout. Visual design.                          |
| **[[H4.8 Project: Contact Form]]**  | Building complete contact form; name, email, message; validation; accessibility; responsive layout; submission handling. Real form.      |
