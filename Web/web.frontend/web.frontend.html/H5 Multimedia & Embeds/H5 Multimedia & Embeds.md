# H5 Multimedia & Embeds

Master embedding multimedia content in HTML. Learn audio, video, iframes, and responsive media. Understand formats, codecs, accessibility, and optimization. Multimedia enriches user experience and is essential for modern, engaging websites.

| Topic                              | Focus & Purpose                                                                                                                                  |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **[[H5.1 Images Deep Dive]]**       | Image formats (JPEG, PNG, WebP, SVG); optimization; responsive images; srcset and sizes; picture element; lazy loading. Optimized images.        |
| **[[H5.2 Audio Element]]**          | audio tag; source element; controls; autoplay; loop; preload; audio formats; fallback content; accessibility. Sound.                             |
| **[[H5.3 Video Element]]**          | video tag; controls; poster; multiple sources; codecs; video formats; accessibility; captions and subtitles. Video content.                      |
| **[[H5.4 Track Element]]**          | Subtitles and captions; VTT format; multiple tracks; kind attribute; srclang; accessibility; internationalization. Text tracks.                  |
| **[[H5.5 SVG Graphics]]**           | Inline SVG; SVG elements; scalable graphics; SVG vs images; animation; accessibility; use cases; optimization. Vector graphics.                  |
| **[[H5.6 Canvas Element]]**         | canvas basics; 2D context; drawing shapes; images; text; animations; interactive graphics; use cases. Programmatic graphics.                     |
| **[[H5.7 iframe & Embeds]]**        | iframe element; embedding external content; YouTube videos; maps; social media; security considerations; responsive embeds. Third-party content. |
| **[[H5.8 Project: Media Gallery]]** | Building image gallery; video player; audio playlist; responsive design; accessibility; lazy loading; optimization. Multimedia showcase.         |
