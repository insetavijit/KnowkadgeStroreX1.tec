# H9 Accessibility (a11y)

Master web accessibility (a11y) in HTML. Learn WCAG guidelines, screen reader support, keyboard navigation, and inclusive design. Build websites that work for everyone. Accessibility is a legal requirement and moral imperative.

| Topic                                   | Focus & Purpose                                                                                                                         |
| --------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **[[H9.1 Accessibility Fundamentals]]**  | What is a11y; WCAG guidelines; POUR principles; legal requirements; assistive technologies; why accessibility matters. Foundation.      |
| **[[H9.2 Semantic HTML for A11y]]**      | Using proper elements; headings hierarchy; landmarks; semantic structure; screen reader navigation; meaningful markup. Accessible base. |
| **[[H9.3 Keyboard Navigation]]**         | Tabindex; focus management; skip links; keyboard accessibility; focus indicators; interactive elements. Keyboard users.                 |
| **[[H9.4 ARIA Roles & Properties]]**     | ARIA roles; states and properties; live regions; aria-label patterns; when to use ARIA; ARIA best practices. Enhanced accessibility.    |
| **[[H9.5 Forms Accessibility]]**         | Accessible labels; error messages; validation feedback; form instructions; required fields; error recovery. Inclusive forms.            |
| **[[H9.6 Media Accessibility]]**         | Alt text best practices; captions and transcripts; audio descriptions; accessible images; accessible video. Inclusive media.            |
| **[[H9.7 Testing Accessibility]]**       | Screen reader testing; axe DevTools; WAVE; Lighthouse; keyboard testing; accessibility audits; automated testing. Quality assurance.    |
| **[[H9.8 Project: Accessible Website]]** | Building fully accessible site; WCAG AA compliance; screen reader optimization; keyboard navigation; comprehensive a11y. Gold standard. |
