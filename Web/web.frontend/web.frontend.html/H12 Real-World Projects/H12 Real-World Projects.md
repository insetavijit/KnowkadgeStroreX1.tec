# H12 Real-World Projects

Master building production HTML websites. Learn complete project workflows, from planning to deployment. Build portfolio pieces across different use cases. Real projects solidify skills and create portfolio work.

| Topic                                 | Focus & Purpose                                                                                                                       |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **[[H12.1 Personal Portfolio]]**       | Portfolio structure; about page; projects showcase; contact section; resume; responsive design; deployment. Professional presence.    |
| **[[H12.2 Business Landing Page]]**    | Hero section; features; testimonials; call-to-action; contact form; SEO optimization; conversion-focused. Business website.           |
| **[[H12.3 Blog Homepage]]**            | Blog layout; article listings; sidebar; pagination; tag system; RSS feed; reading experience; content structure. Publishing platform. |
| **[[H12.4 E-commerce Product Page]]**  | Product showcase; image gallery; product details; add to cart form; reviews section; related products; microdata. Online store.       |
| **[[H12.5 Documentation Site]]**       | Docs structure; navigation; code examples; syntax highlighting; search functionality; version switching. Technical docs.              |
| **[[H12.6 Event Landing Page]]**       | Event details; schedule; speakers; registration form; countdown; location map; social proof. Event promotion.                         |
| **[[H12.7 Restaurant Website]]**       | Menu display; gallery; reservations form; hours and location; reviews; mobile-friendly; local SEO. Local business.                    |
| **[[H12.8 Complete Multi-Page Site]]** | Full website with navigation; multiple pages; consistent design; forms; media; SEO; accessibility; deployment. Production project.    |
