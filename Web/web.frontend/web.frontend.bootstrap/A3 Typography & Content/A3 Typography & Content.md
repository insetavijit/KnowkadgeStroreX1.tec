# A3 Typography & Content

Master Bootstrap typography for professional, readable content. Learn headings, body text, lists, and text utilities. Build accessible, well-structured content that enhances user experience. Typography skills are essential for content-rich applications.

|Topic|Focus & Purpose|
|---|---|
|**[[A3.1 Headings & Display]]**|h1-h6 headings; display headings (display-1 to display-6); heading customization; semantic headings. Visual hierarchy.|
|**[[A3.2 Body Text & Lead]]**|Paragraph styling; lead class; mark; small; del; ins; abbreviations; text formatting. Content styling.|
|**[[A3.3 Text Utilities]]**|Text alignment; text wrapping; text transform; font weight; font style; line height; monospace. Text control.|
|**[[A3.4 Lists & Blockquotes]]**|Unstyled lists; inline lists; description lists; blockquotes; blockquote footers; source attribution. Structured content.|
|**[[A3.5 Color Utilities]]**|Text colors (text-primary, text-secondary); background colors; contextual colors; opacity utilities. Color system.|
|**[[A3.6 Responsive Typography]]**|Responsive font sizes (fs-1 to fs-6); viewport-based scaling; readable line lengths. Adaptive typography.|
|**[[A3.7 Icons & Iconography]]**|Bootstrap Icons library; icon sizing; icon colors; icon alignment; Font Awesome integration; SVG icons. Icon usage.|
|**[[A3.8 Project - Blog Article Page]]**|Building article layout; typography hierarchy; blockquotes; images; lists; responsive text. Content showcase.|
