# A4 Buttons & Interactive Elements

Master Bootstrap buttons and interactive components. Learn button variants, sizes, groups, and states. Build engaging user interfaces with proper feedback mechanisms. Button mastery is crucial for interactive applications and user experience.

|Topic|Focus & Purpose|
|---|---|
|**[[A4.1 Button Basics]]**|Button classes; button variants (primary, secondary, success, danger, warning, info, light, dark); accessibility. Core buttons.|
|**[[A4.2 Button Sizes & States]]**|Button sizes (btn-lg, btn-sm); block buttons; disabled state; active state; loading states. Button variations.|
|**[[A4.3 Outline Buttons]]**|Outline button variants; when to use outline; hover effects; outline vs solid buttons; design principles. Alternative styling.|
|**[[A4.4 Button Groups]]**|Button group components; toolbar; sizing; vertical groups; split buttons; nested groups. Grouped actions.|
|**[[A4.5 Button Dropdowns]]**|Dropdown buttons; split button dropdowns; dropdown directions; dropdown menus; keyboard navigation. Dropdown interactions.|
|**[[A4.6 Close Button]]**|Close button component; dismissible alerts; modal close; accessibility; styling; dark variant. Dismiss actions.|
|**[[A4.7 Link Buttons]]**|Anchor as buttons; link styling; stretched links; icon buttons; button vs link semantics. Link interactions.|
|**[[A4.8 Project - Action Panel]]**|Building action interface; button combinations; button groups; dropdowns; responsive button layouts. Interactive showcase.|
