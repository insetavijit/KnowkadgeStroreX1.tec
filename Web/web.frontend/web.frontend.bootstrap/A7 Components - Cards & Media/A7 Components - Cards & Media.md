# A7 Components - Cards & Media

Master Bootstrap cards and media components for content presentation. Learn card layouts, image handling, and content organization. Build visually appealing content displays used in modern web applications.

|Topic|Focus & Purpose|
|---|---|
|**[[A7.1 Card Basics]]**|Card structure; card body; titles; text; links; images; card sizing; card borders. Core card component.|
|**[[A7.2 Card Header & Footer]]**|Header and footer sections; navigation in cards; tabbed cards; card links; card actions. Card structure.|
|**[[A7.3 Card Images]]**|Image caps (top, bottom); image overlays; card backgrounds; horizontal cards; responsive images. Visual cards.|
|**[[A7.4 Card Layouts]]**|Card groups; card decks (deprecated alternatives); card grid; masonry layout; responsive cards. Card arrangements.|
|**[[A7.5 List Groups]]**|List group basics; active and disabled; links and buttons; flush list groups; numbered; horizontal. Grouped content.|
|**[[A7.6 List Group + Card]]**|Cards with list groups; actionable list groups; badges in lists; contextual colors; checkboxes. Combined components.|
|**[[A7.7 Figures]]**|Figure component; figcaption; image alignment; responsive figures; caption styling. Image with context.|
|**[[A7.8 Project - Product Catalog]]**|Building product grid; cards; images; pricing; hover effects; responsive layout; filtering UI. E-commerce showcase.|
