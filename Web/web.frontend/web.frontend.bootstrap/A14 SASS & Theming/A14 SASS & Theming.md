# A14 SASS & Theming

Master Bootstrap SASS customization for branded designs. Learn variables, maps, mixins, and theme creation. Build custom design systems on Bootstrap foundation. Theming skills are essential for agency and enterprise work.

|Topic|Focus & Purpose|
|---|---|
|**[[A14.1 SASS Variables]]**|Default variables; variable overrides; color variables; spacing variables; typography variables. Customization foundation.|
|**[[A14.2 SASS Maps]]**|Color maps; theme colors map; adding/removing colors; map-get; map-merge; custom maps. Advanced customization.|
|**[[A14.3 Mixins & Functions]]**|Bootstrap mixins; responsive breakpoints mixin; color functions; utility generation; custom mixins. SASS tooling.|
|**[[A14.4 Component Overrides]]**|Component variable overrides; button customization; form styling; navbar theming; card styling. Component theming.|
|**[[A14.5 Color Modes]]**|Bootstrap 5.3+ color modes; dark mode; custom color modes; mode switching; CSS custom properties. Theme switching.|
|**[[A14.6 Custom Utilities]]**|Utility API; generating utilities; modifying utilities; print utilities; custom utility classes. Extended utilities.|
|**[[A14.7 Build System]]**|SASS compilation; build tools (npm scripts, Gulp, Webpack); watch mode; production builds. Development workflow.|
|**[[A14.8 Project - Custom Theme]]**|Building branded theme; color scheme; custom components; dark mode; responsive typography. Theming showcase.|
