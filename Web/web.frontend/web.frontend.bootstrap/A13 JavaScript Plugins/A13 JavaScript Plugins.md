# A13 JavaScript Plugins

Master Bootstrap JavaScript components. Learn data attributes, JavaScript API, and event handling. Build dynamic, interactive interfaces without jQuery. JavaScript knowledge is essential for interactive Bootstrap applications.

|Topic|Focus & Purpose|
|---|---|
|**[[A13.1 Data Attributes API]]**|Data-bs-* attributes; toggle behaviors; target specification; no JavaScript initialization. Declarative approach.|
|**[[A13.2 JavaScript API]]**|JavaScript initialization; constructor methods; getInstance; getOrCreateInstance; options. Programmatic control.|
|**[[A13.3 Events]]**|Bootstrap events; show/shown; hide/hidden; event listeners; preventing default; custom handlers. Component lifecycle.|
|**[[A13.4 Methods]]**|Component methods (show, hide, toggle); dispose; update; handleUpdate; async methods. Runtime control.|
|**[[A13.5 Dropdown JavaScript]]**|Dropdown via JS; dynamic dropdowns; dropdown events; custom behaviors; programmatic toggle. Dropdown programming.|
|**[[A13.6 Modal JavaScript]]**|Modal via JS; dynamic modals; passing data; nested modals; modal events; modal stacking. Modal programming.|
|**[[A13.7 Tab JavaScript]]**|Tab via JS; dynamic tabs; tab events; programmatic switching; deep linking; history. Tab programming.|
|**[[A13.8 Project - Dynamic Interface]]**|Building interactive app; JavaScript-driven components; dynamic content; events; animations. JS showcase.|
