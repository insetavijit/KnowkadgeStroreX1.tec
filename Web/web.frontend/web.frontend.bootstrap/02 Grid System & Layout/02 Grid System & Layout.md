# 02 Grid System & Layout

Master Bootstrap's powerful 12-column grid system for creating responsive layouts. Learn containers, rows, columns, breakpoints, and advanced grid features. Grid mastery is fundamental for all Bootstrap development and tested in every frontend job interview.

|Topic|Focus & Purpose|
|---|---|
|**[[2.1 Container System]]**|Container types (container, container-fluid, container-{breakpoint}); max-widths; padding; when to use each. Layout foundation.|
|**[[2.2 Grid Fundamentals]]**|12-column system; row and column relationship; column sizing; auto-layout columns; equal-width columns. Core grid concepts.|
|**[[2.3 Responsive Breakpoints]]**|xs, sm, md, lg, xl, xxl breakpoints; breakpoint classes; mobile-first approach; responsive column sizing. Multi-device layouts.|
|**[[2.4 Column Ordering]]**|Order classes; visual reordering; responsive order changes; order-first; order-last; layout control. Flexible layouts.|
|**[[2.5 Offset & Alignment]]**|Column offsets; horizontal alignment (justify-content); vertical alignment (align-items); gap utilities. Precise positioning.|
|**[[2.6 Nesting & Gutters]]**|Nested grids; multi-level layouts; gutter customization; gutter utilities; removing gutters. Complex layouts.|
|**[[2.7 CSS Grid & Flexbox]]**|Bootstrap + CSS Grid; d-grid utilities; gap classes; Flexbox utilities; modern layout techniques. Advanced layout.|
|**[[2.8 Project: Dashboard Layout]]**|Building responsive dashboard; sidebar; main content; responsive grid; collapsible navigation. Production layout.|
