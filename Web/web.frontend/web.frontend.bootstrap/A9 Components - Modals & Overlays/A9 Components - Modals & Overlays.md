# A9 Components - Modals & Overlays

Master Bootstrap modals and overlay components. Learn dialogs, tooltips, popovers, and offcanvas. Build sophisticated interactive interfaces. Modal mastery is critical for modern web applications and user workflows.

|Topic|Focus & Purpose|
|---|---|
|**[[A9.1 Modal Basics]]**|Modal structure; modal triggers; modal body; modal footer; static backdrop; keyboard dismissal. Dialog windows.|
|**[[A9.2 Modal Sizes & Options]]**|Modal sizes (sm, lg, xl); fullscreen modals; centered modals; scrollable content; long content. Modal variations.|
|**[[A9.3 Modal Events]]**|Show/hide events; JavaScript control; dynamic content; preventing close; custom triggers. Modal interaction.|
|**[[A9.4 Tooltips]]**|Tooltip component; placement; triggers; HTML content; disabled elements; tooltip styling. Contextual hints.|
|**[[A9.5 Popovers]]**|Popover component; placement; triggers; dismissible; title and content; dynamic content. Rich tooltips.|
|**[[A9.6 Offcanvas]]**|Offcanvas component; placement; backdrop; scrolling; responsive offcanvas; navigation panels. Slide-out panels.|
|**[[A9.7 Collapse]]**|Collapse component; accordion; multiple targets; horizontal collapse; toggle controls. Expandable content.|
|**[[A9.8 Project - Interactive Dashboard]]**|Building dashboard modals; confirmation dialogs; settings panel; tooltips; offcanvas menu. Overlay showcase.|
