# A6 Forms & Inputs

Master Bootstrap forms for user data collection. Learn form controls, validation, layouts, and accessibility. Build production-ready forms with excellent UX. Form skills are essential for interactive applications and tested in job interviews.

|Topic|Focus & Purpose|
|---|---|
|**[[A6.1 Form Controls]]**|Input types (text, email, password, number, date); textarea; select; file input; readonly; disabled. Input elements.|
|**[[A6.2 Form Layouts]]**|Vertical forms; horizontal forms; inline forms; form groups; labels; form grid; column sizing. Form structure.|
|**[[A6.3 Checkboxes & Radios]]**|Checkbox styling; radio buttons; inline options; switches; toggle states; button groups as input. Selection inputs.|
|**[[A6.4 Input Groups]]**|Input group addons; prepend and append; button addons; dropdown addons; sizing; validation integration. Enhanced inputs.|
|**[[A6.5 Floating Labels]]**|Floating label forms; animation effects; placeholder requirements; accessibility considerations; styling. Modern form UX.|
|**[[A6.6 Range & Color]]**|Range slider; custom range; color picker; number input styling; specialized inputs. Special inputs.|
|**[[A6.7 Form Validation]]**|Client-side validation; server-side validation; validation styles; custom messages; novalidate. Data integrity.|
|**[[A6.8 Project - Contact Form]]**|Building complete form; validation; accessibility; floating labels; input groups; responsive layout. Form showcase.|
