# 12 Utilities & Helpers

Master Bootstrap utility classes for rapid development. Learn spacing, display, flexbox, and positioning utilities. Build layouts efficiently with utility-first approach. Utility mastery dramatically increases development speed.

|Topic|Focus & Purpose|
|---|---|
|**[[12.1 Spacing Utilities]]**|Margin (m-*); padding (p-*); spacing scale; responsive spacing; negative margins; gap utilities. Whitespace control.|
|**[[12.2 Display Utilities]]**|Display classes (d-none, d-block, d-flex, d-grid); responsive display; visibility utilities. Element visibility.|
|**[[12.3 Flexbox Utilities]]**|Flex direction; justify-content; align-items; align-self; flex-wrap; flex-grow/shrink; order. Flex layout.|
|**[[12.4 Position Utilities]]**|Position classes; top/bottom/start/end; translate; centering; sticky positioning; z-index. Element positioning.|
|**[[12.5 Sizing Utilities]]**|Width and height classes (w-*, h-*); max-width; min-width; viewport sizing; auto sizing. Dimension control.|
|**[[12.6 Border Utilities]]**|Border classes; border colors; border radius; rounded corners; border width; subtractive borders. Edge styling.|
|**[[12.7 Shadow & Opacity]]**|Box shadows; shadow sizes; opacity utilities; background opacity; transparency effects. Visual depth.|
|**[[12.8 Project: Utility-First Page]]**|Building page with utilities; minimal custom CSS; utility composition; responsive design. Efficiency showcase.|
