# 05 Navigation Components

Master Bootstrap navigation for intuitive site navigation. Learn navbars, navs, breadcrumbs, and pagination. Build user-friendly navigation systems essential for all websites. Navigation skills are required for every frontend position.

|**[[5.4 Nav Component]]**|Base nav; tabs; pills; fill and justify; vertical navigation; accessible navigation. Secondary navigation.|
|**[[5.5 Breadcrumbs]]**|Breadcrumb component; dividers; dynamic breadcrumbs; SEO benefits; accessibility; styling. Location awareness.|
|**[[5.6 Pagination]]**|Pagination component; sizing; alignment; disabled and active states; icons; accessibility. Content navigation.|
|**[[5.7 Scrollspy]]**|Scrollspy plugin; navigation highlighting; smooth scrolling; offset configuration; single-page apps. Scroll navigation.|
|**[[5.8 Project: Multi-Section Website]]**|Building complete navigation; responsive navbar; sections; scrollspy; breadcrumbs; footer navigation. Navigation showcase.|
