# R11 Performance Optimization

Master React performance optimization techniques. Learn memoization, profiling, and rendering optimization. Performance skills are critical for user experience and frequently tested in senior interviews.

|Topic|Focus & Purpose|
|---|---|

|**[[R11.2 Profiling]]**|React DevTools Profiler; flamegraphs; identifying bottlenecks; render timing. Performance analysis.|
|**[[R11.3 React.memo]]**|Component memoization; when to use memo; comparison function; memo pitfalls. Component optimization.|
|**[[R11.4 useMemo & useCallback]]**|Value memoization; function memoization; dependency management; optimization patterns. Hook optimization.|
|**[[R11.5 List Optimization]]**|Virtualization; react-window; react-virtualized; large list handling; windowing. List performance.|
|**[[R11.6 Code Splitting]]**|Route-based splitting; component splitting; preloading; bundle analysis; lazy loading. Bundle optimization.|
|**[[R11.7 Image Optimization]]**|Image loading; lazy loading images; responsive images; formats; compression. Asset optimization.|
|**[[R11.8 Project: Performance Audit]]**|Optimizing React app; profiling; implementing fixes; measuring improvements. Optimization project.|
