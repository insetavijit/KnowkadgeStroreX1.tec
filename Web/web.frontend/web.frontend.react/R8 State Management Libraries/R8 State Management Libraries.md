# R8 State Management Libraries

Master global state management for complex applications. Learn Redux Toolkit, Zustand, and React Query. State management skills are critical for large-scale applications and frequently tested in interviews.

|Topic|Focus & Purpose|
|---|---|
|**[[R8.1 When to Use Global State]]**|State categories; local vs global; state lifting limits; when to add libraries; over-engineering. Decision making.|
|**[[R8.2 Redux Toolkit]]**|Redux concepts; createSlice; configureStore; useSelector; useDispatch; Redux DevTools. Industry standard.|
|**[[R8.3 Redux Async]]**|createAsyncThunk; loading states; error handling; RTK Query basics; cache management. Async Redux.|
|**[[R8.4 Zustand]]**|Zustand setup; creating stores; selectors; actions; middleware; persist; subscribe. Lightweight alternative.|
|**[[R8.5 React Query]]**|TanStack Query; useQuery; useMutation; caching; invalidation; optimistic updates. Server state.|
|**[[R8.6 Context + useReducer]]**|Combining Context and useReducer; provider pattern; action dispatch; scaling. Built-in solution.|
|**[[R8.7 State Architecture]]**|State organization; slice design; normalization; selectors; performance; migrations. Architecture patterns.|
|**[[R8.8 Project: E-Commerce State]]**|Building cart system; Redux Toolkit; async operations; persistence; state sync. State project.|
