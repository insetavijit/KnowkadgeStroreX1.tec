# R2 State Management Fundamentals

Master React state for building interactive applications. Learn useState, state updates, and component lifecycle. State management is the core concept for React interactivity and tested in every frontend interview.

| Topic                                  | Focus & Purpose                                                                                                              |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **[[R2.1 useState Hook]]**              | useState basics; state initialization; state updates; functional updates; state immutability. Local state.                   |
| **[[R2.2 State Updates]]**              | Updating primitives; updating objects; updating arrays; spread operator; avoiding mutations; batch updates. Proper updates.  |
| **[[R2.3 Lifting State Up]]**           | Shared state; state lifting; prop drilling; component communication; state ownership. State architecture.                    |
| **[[R2.4 Controlled Components]]**      | Form inputs; controlled vs uncontrolled; onChange handlers; form state; input binding. Form handling.                        |
| **[[R2.5 Event Handling]]**             | Event handlers; synthetic events; event binding; event parameters; preventing defaults; event delegation. User interactions. |
| **[[R2.6 Conditional Rendering]]**      | If/else; ternary operator; && operator; switch statements; null rendering; loading states. Dynamic UI.                       |
| **[[R2.7 Lists & Keys]]**               | Rendering lists; map function; key prop; key importance; stable keys; index as key pitfalls. List rendering.                 |
| **[[R2.8 Project - Todo Application]]** | Building todo app; CRUD operations; state management; event handling; local storage. Interactive project.                    |
