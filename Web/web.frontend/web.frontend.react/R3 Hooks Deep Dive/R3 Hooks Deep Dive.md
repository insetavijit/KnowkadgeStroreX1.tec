# R3 Hooks Deep Dive

Master React Hooks for modern functional components. Learn useEffect, useRef, useContext, and custom hooks. Hooks knowledge is essential for all React development and the foundation for advanced patterns.

|Topic|Focus & Purpose|
|---|---|
|**[[R3.1 useEffect Hook]]**|Side effects; effect function; dependency array; cleanup; mounting/unmounting; effect timing. Side effect management.|
|**[[R3.2 useEffect Patterns]]**|Data fetching; subscriptions; DOM manipulation; event listeners; timers; effect organization. Common patterns.|
|**[[R3.3 useRef Hook]]**|Refs basics; DOM access; persisting values; ref vs state; forward refs; callback refs. Reference management.|
|**[[R3.4 useContext Hook]]**|Context API; creating context; Provider; Consumer; useContext; avoiding prop drilling. Global state.|
|**[[R3.5 useMemo & useCallback]]**|Memoization; useMemo for values; useCallback for functions; dependency arrays; optimization strategies. Performance hooks.|
|**[[R3.6 useReducer Hook]]**|Reducer pattern; dispatch; actions; complex state; useReducer vs useState; reducer composition. Advanced state.|
|**[[R3.7 Custom Hooks]]**|Creating custom hooks; hook composition; reusable logic; hook conventions; sharing state logic. Code reuse.|
|**[[R3.8 Project: Data Dashboard]]**|Building dashboard; multiple hooks; data fetching; custom hooks; performance optimization. Hooks project.|
