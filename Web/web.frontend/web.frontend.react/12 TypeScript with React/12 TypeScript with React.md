# 12 TypeScript with React

Master TypeScript for type-safe React development. Learn component types, generic components, and type patterns. TypeScript is increasingly required for professional React positions.

|**[[12.3 Hook Types]]**|useState types; useRef types; useReducer types; custom hook types. Typed hooks.|
|**[[12.4 Event Types]]**|Event handler types; synthetic events; form events; mouse events; keyboard events. Event typing.|
|**[[12.5 Generic Components]]**|Generic props; generic constraints; reusable typed components; type inference. Advanced typing.|
|**[[12.6 API Types]]**|Response types; Zod inference; type generation; API type safety. Data typing.|
|**[[12.7 Advanced Patterns]]**|Discriminated unions; template literals; mapped types; utility types; strict patterns. Type patterns.|
|**[[12.8 Project: Typed Application]]**|Building fully typed app; strict TypeScript; type coverage; best practices. TypeScript project.|
