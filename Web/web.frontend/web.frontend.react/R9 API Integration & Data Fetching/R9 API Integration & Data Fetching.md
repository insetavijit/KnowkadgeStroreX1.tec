# R9 API Integration & Data Fetching

Master data fetching patterns for React applications. Learn fetch, axios, React Query, and error handling. API integration skills are essential for connecting React frontends to backends.

|Topic|Focus & Purpose|
|---|---|
|**[[R9.1 Fetch API]]**|Native fetch; GET/POST requests; headers; JSON handling; error handling. Built-in fetching.|
|**[[R9.2 Axios]]**|Axios setup; request methods; interceptors; request/response transformation; cancellation. Popular library.|
|**[[R9.3 useEffect Data Fetching]]**|Fetching in useEffect; loading states; error states; cleanup; race conditions. Hook-based fetching.|
|**[[R9.4 React Query Patterns]]**|Query patterns; mutation patterns; infinite queries; pagination; prefetching; caching strategies. TanStack Query.|
|**[[R9.5 SWR]]**|SWR library; stale-while-revalidate; mutations; global configuration; comparison with React Query. Alternative library.|
|**[[R9.6 Error Handling]]**|Error boundaries for data; retry logic; error states; user feedback; fallback UI. Robust error handling.|
|**[[R9.7 Loading States]]**|Loading indicators; skeleton screens; suspense for data; progressive loading; optimistic UI. UX patterns.|
|**[[R9.8 Project: API Dashboard]]**|Building data dashboard; multiple APIs; caching; error handling; real-time updates. Data project.|
