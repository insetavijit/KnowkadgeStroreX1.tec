# N9 Forms & Data Validation

Master form handling with Server Actions, React Hook Form, and validation libraries. Learn progressive enhancement, optimistic updates, and error handling. Build production-grade forms with excellent UX. Form mastery is critical for interactive applications.

|Topic|Focus & Purpose|
|---|---|
|**[[N9.1 Server Actions for Forms]]**|Server Actions basics; form actions; FormData; progressive enhancement; no-JS forms; revalidation. Modern form handling.|
|**[[N9.2 React Hook Form]]**|React Hook Form setup; useForm hook; controlled inputs; field validation; form submission; performance. Client-side forms.|
|**[[N9.3 Zod Validation]]**|Zod schema; type inference; validation rules; error messages; server-side validation; client-side validation. Type-safe validation.|
|**[[N9.4 Form UI Components]]**|Form inputs; select dropdowns; checkboxes; radio buttons; file uploads; accessible forms. Form elements.|
|**[[N9.5 Optimistic Updates]]**|useOptimistic hook; optimistic UI; rollback handling; pending states; user experience. Instant feedback.|
|**[[N9.6 File Uploads]]**|File upload handling; image uploads; cloud storage (S3, Cloudinary); validation; progress indicators. Handling files.|
|**[[N9.7 Multi-Step Forms]]**|Wizard forms; step management; data persistence; progress indicators; validation per step. Complex forms.|
|**[[N9.8 Project: Contact & Onboarding]]**|Building contact form; multi-step onboarding; validation; Server Actions; file uploads; email integration. Production forms.|
