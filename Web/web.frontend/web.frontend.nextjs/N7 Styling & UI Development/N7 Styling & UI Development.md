# N7 Styling & UI Development

Master modern styling approaches for Next.js. Learn Tailwind CSS, CSS Modules, CSS-in-JS, and component libraries. Build beautiful, responsive UIs efficiently. Styling skills are essential for front-end focused Next.js positions.

|Topic|Focus & Purpose|
|---|---|
|**[[N7.1 Tailwind CSS Integration]]**|Tailwind setup; utility classes; responsive design; dark mode; customization; best practices. Most popular approach.|
|**[[N7.2 CSS Modules]]**|CSS Modules basics; scoped styles; composition; naming conventions; when to use CSS Modules. Traditional approach.|
|**[[N7.3 Component Libraries]]**|shadcn/ui; Material-UI; Chakra UI; Radix UI; component integration; customization. Pre-built components.|
|**[[N7.4 Responsive Design]]**|Mobile-first design; breakpoints; responsive utilities; container queries; modern responsive patterns. Cross-device support.|
|**[[N7.5 Dark Mode]]**|Dark mode implementation; next-themes; system preference; toggle UI; persistence; color schemes. Theme switching.|
|**[[N7.6 Animation Libraries]]**|Framer Motion; animation patterns; page transitions; scroll animations; interactive elements. Motion design.|
|**[[N7.7 Icon Libraries]]**|Lucide React; Heroicons; React Icons; icon usage; optimization; best practices. Icon integration.|
|**[[N7.8 Project: Modern Dashboard UI]]**|Building dashboard; Tailwind; shadcn/ui; dark mode; responsive design; animations; charts. Complete UI project.|
