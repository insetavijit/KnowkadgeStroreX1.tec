# WP04 Theme Design & Frontend

Master modern frontend development for WordPress themes. Learn Sass/SCSS, responsive design, and CSS frameworks. Build beautiful, professional designs. Frontend skills differentiate developers and are crucial for agency and client work.

| Topic                                        | Focus & Purpose                                                                                                                      |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **[[WP4.1 Sass/SCSS Integration]]**            | Sass basics; variables; nesting; mixins; partials; compilation; Sass in WordPress; modern CSS workflow. CSS preprocessing.           |
| **[[WP4.2 CSS Frameworks]]**                   | Bootstrap integration; Foundation; Tailwind CSS; framework selection; customization; utility-first CSS. Framework development.        |
| **[[WP4.3 Responsive Design]]**                | Mobile-first approach; media queries; breakpoints; fluid layouts; responsive images; touch-friendly design. Cross-device compatibility.|
| **[[WP4.4 CSS Grid & Flexbox]]**               | Modern layout systems; grid layouts; flexbox patterns; complex layouts; browser support; practical layouts. Advanced CSS.             |
| **[[WP4.5 Typography & Web Fonts]]**           | Google Fonts; Adobe Fonts; font loading; typography systems; readability; performance; font management. Text styling.                |
| **[[WP4.6 JavaScript & jQuery]]**              | Enqueuing scripts; jQuery in WordPress; vanilla JS; ES6+ features; JavaScript organization; performance. Interactive features.        |
| **[[WP4.7 Animation & Interactions]]**         | CSS animations; transitions; JavaScript animations; scroll effects; hover states; micro-interactions. Engaging UX.                    |
| **[[WP4.8 Project: Modern Business Theme]]**   | Professional design; responsive layout; custom CSS; JavaScript features; complete frontend; production-ready design. Design showcase.  |
