# Data Lineage

```python
class DataLineage:
    def record_source(self, data_id: str, source: str, metadata: dict):
        lineage_db.insert({
            "data_id": data_id,
            "source": source,
            "metadata": metadata,
            "timestamp": datetime.utcnow()
        })
    
    def record_transformation(self, input_id: str, output_id: str, operation: str):
        lineage_db.insert({
            "input": input_id,
            "output": output_id,
            "operation": operation,
            "timestamp": datetime.utcnow()
        })
    
    def get_lineage(self, data_id: str) -> dict:
        return lineage_db.query_graph(data_id)
```

**Quick Ref:** Track data source, transformations, and flow for auditability.
