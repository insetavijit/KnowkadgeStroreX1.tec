## 2. Contract Analysis

```python
class ContractAnalyzer:
    def __init__(self):
        self.llm = ChatOpenAI(model="gpt-4o")
    
    def analyze_contract(self, contract_text: str) -> dict:
        prompt = f"""
Analyze this contract:

{contract_text}

Extract:
1. Parties involved
2. Key obligations for each party
3. Important dates and deadlines
4. Payment terms
5. Termination clauses
6. Potential risks or unusual provisions
7. Missing standard clauses

Format as structured JSON.
"""
        response = self.llm.invoke(prompt)
        return parse_json(response.content)
    
    def find_risks(self, contract_text: str) -> list:
        prompt = """
Identify potential legal risks in this contract:
- Unfavorable terms
- Ambiguous language
- Missing protections
- Unusual liabilities
"""
        return self.llm.invoke(prompt).content
```

## 3. Legal Research

```python
class LegalResearchAssistant:
    def __init__(self, case_law_vectorstore):
        self.llm = ChatOpenAI(model="gpt-4o")
        self.vectorstore = case_law_vectorstore
    
    def research_issue(self, legal_issue: str) -> dict:
        # Find relevant cases
        relevant_cases = self.vectorstore.similarity_search(
            legal_issue, 
            k=10
        )
        
        # Analyze with context
        prompt = f"""
Legal Issue: {legal_issue}

Relevant Cases:
{format_cases(relevant_cases)}

Provide:
1. Summary of applicable law
2. Key precedents
3. How courts have ruled on similar issues
4. Potential arguments for both sides
"""
        
        analysis = self.llm.invoke(prompt)
        
        return {
            "analysis": analysis.content,
            "cited_cases": [c.metadata["citation"] for c in relevant_cases]
        }
```

## 5. Legal Limitations

**AI CANNOT**:
| Prohibited | Why |
|------------|-----|
| **Practice law** | Unauthorized practice of law |
| **Give legal advice** | Requires bar admission |
| **Make strategy decisions** | Professional judgment required |
| **Represent clients** | Attorney-client relationship |

**Proper Disclaimers**:
```python
LEGAL_DISCLAIMER = """
⚠️ LEGAL DISCLAIMER:
This is NOT legal advice. This information is provided for 
educational/research purposes only. For legal matters, 
consult a licensed attorney in your jurisdiction.
"""

def legal_response(response: str) -> str:
    return f"{response}\n\n{LEGAL_DISCLAIMER}"
```

**Use Cases by Risk**:
| OK | Requires Review | Avoid |
|-----|-----------------|-------|
| Research summaries | Contract drafts | Legal advice |
| Case finding | Document templates | Strategy |
| Plain language | Risk flags | Representation |
