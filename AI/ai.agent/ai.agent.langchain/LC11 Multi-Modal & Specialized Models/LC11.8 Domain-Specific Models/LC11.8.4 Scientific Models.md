## 2. Paper Understanding

```python
class PaperAnalyzer:
    def __init__(self):
        self.llm = ChatOpenAI(model="gpt-4o")
    
    def summarize_paper(self, paper_text: str) -> dict:
        prompt = f"""
Summarize this scientific paper:

{paper_text}

Provide:
1. One-paragraph summary
2. Key findings (bullet points)
3. Methodology overview
4. Limitations mentioned
5. Practical implications
6. Key terms explained
"""
        return self.llm.invoke(prompt).content
    
    def explain_for_audience(self, paper_text: str, audience: str) -> str:
        prompt = f"""
Explain this paper for a {audience}:

{paper_text}

Use appropriate level of technical detail.
"""
        return self.llm.invoke(prompt).content

# Usage
analyzer = PaperAnalyzer()
summary = analyzer.explain_for_audience(paper, "undergraduate student")
summary = analyzer.explain_for_audience(paper, "domain expert")
```

## 4. Research Assistance

```python
class ResearchAssistant:
    def __init__(self, paper_vectorstore):
        self.llm = ChatOpenAI(model="gpt-4o")
        self.vectorstore = paper_vectorstore
    
    def literature_review(self, topic: str) -> str:
        """Generate literature review on topic."""
        relevant_papers = self.vectorstore.similarity_search(topic, k=20)
        
        prompt = f"""
Write a literature review section on: {topic}

Using these papers as sources:
{format_papers(relevant_papers)}

Include:
1. Overview of the field
2. Key developments chronologically
3. Current state of research
4. Open questions
5. Proper citations (Author, Year)
"""
        return self.llm.invoke(prompt).content
    
    def find_gaps(self, papers: list) -> str:
        """Identify research gaps from literature."""
        prompt = """
Based on these papers, identify:
1. Unexplored research questions
2. Methodological limitations to address
3. Conflicting findings to resolve
4. Suggested future directions
"""
        return self.llm.invoke(prompt).content
    
    def suggest_methodology(self, research_question: str) -> str:
        """Suggest research methodology."""
        prompt = f"""
For this research question: {research_question}

Suggest:
1. Appropriate research design
2. Data collection methods
3. Analysis techniques
4. Potential challenges
5. Required resources
"""
        return self.llm.invoke(prompt).content
```

## 5. Scientific Reasoning Limitations

**What AI Does Well**:
| Good | Why |
|------|-----|
| Literature search | Pattern matching |
| Summarization | Text processing |
| Explanation | Knowledge synthesis |
| Writing polish | Language model strength |

**What AI Struggles With**:
| Struggles | Why |
|-----------|-----|
| Novel discoveries | Trained on existing knowledge |
| Experimental design | Requires tacit expertise |
| Peer review depth | Can miss subtle issues |
| Scientific intuition | No research experience |

**Best Practice**:
```python
# AI as assistant, not oracle
class ScientificWorkflow:
    def analyze(self, data):
        # AI suggests
        ai_analysis = self.llm.invoke(data)
        
        # Human validates
        return {
            "ai_suggestion": ai_analysis,
            "requires_validation": True,
            "next_step": "Expert review"
        }
```
