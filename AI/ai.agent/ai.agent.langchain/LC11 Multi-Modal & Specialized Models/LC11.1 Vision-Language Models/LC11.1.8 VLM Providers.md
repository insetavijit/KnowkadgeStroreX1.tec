# VLM Providers: Choosing the Right Model

Different providers have different strengths for vision tasks.

---

## 1. OpenAI GPT-4o

**LangChain Setup**:
```python
from langchain_openai import ChatOpenAI
llm = ChatOpenAI(model="gpt-4o")
```

| Strength | Weakness |
|----------|----------|
| Fast response time | Cost for high-res images |
| Good general vision | Limited video support |
| Strong tool use | 128k context only |

**Pricing** (approx): $5/1M input tokens, images ~85-680 tokens each.

---

## 2. Anthropic Claude 3

**LangChain Setup**:
```python
from langchain_anthropic import ChatAnthropic
llm = ChatAnthropic(model="claude-3-sonnet-20240229")
```

| Strength | Weakness |
|----------|----------|
| Strong document reasoning | Slower than GPT-4o |
| Good at charts/graphs | 5MB image limit |
| Nuanced analysis | Higher cost (Opus) |

**Models**: Opus (best quality), Sonnet (balanced), Haiku (fast/cheap).

---

## 3. Google Gemini

**LangChain Setup**:
```python
from langchain_google_genai import ChatGoogleGenerativeAI
llm = ChatGoogleGenerativeAI(model="gemini-1.5-pro")
```

| Strength | Weakness |
|----------|----------|
| 1M+ token context | API quota limits |
| Native video support | Slightly less coherent |
| Generous free tier | Gemini API maturity |

Gemini is unique in accepting **video files directly**.

---

## 4. Open Source VLMs

**Self-Hosted Options**:
*   **LLaVA**: Llama + CLIP, runs on consumer GPUs
*   **Qwen-VL**: Alibaba's model, strong Chinese support
*   **CogVLM**: Good for OCR and document understanding

**LangChain Setup (Ollama)**:
```python
from langchain_ollama import ChatOllama
llm = ChatOllama(model="llava:13b")
```

| Strength | Weakness |
|----------|----------|
| No API cost | GPU required |
| Full privacy | Smaller than GPT-4V |
| Customizable | More setup work |

---

## 5. Cost Comparison (Approximate)

| Provider | Cost per 1M Input Tokens | Image Token Count |
|----------|--------------------------|-------------------|
| **GPT-4o** | $5 | 85-680 per image |
| **Claude 3 Sonnet** | $3 | Varies |
| **Gemini 1.5 Pro** | $3.50 | Varies |
| **LLaVA (local)** | $0 | N/A |

**Rule of Thumb**: Processing 1000 images costs $5-50 depending on resolution and provider.
